VIT Rickmate – Campus Ride Sharing Platform
Overview

VIT Rickmate is a campus-focused ride-sharing web application designed to help students find and share rides with others traveling in the same direction.

The platform reduces travel cost, improves convenience, and enables safer commuting within and around campus through peer-based ride matching.

Key Features
Ride creation and discovery within campus
Matching users based on source, destination, and time
Invite-based ride sharing with known users
Real-time communication between users (chat/coordination)
User authentication and secure access
Scalable backend for handling multiple ride requests

Problem Statement

Students often face:

Difficulty finding transport at specific times
High travel costs when traveling individually
Lack of coordination among students going in the same direction

Rickmate solves this by enabling structured ride-sharing within a trusted campus network.

System Workflow
User Authentication
Users sign up/login to access the platform
Ride Creation
A user creates a ride by entering:
Source
Destination
Time
Available seats
Ride Matching
The system identifies users with similar routes and timing
Ride Joining
Other users can request or join available rides
Communication
Users coordinate via in-app interaction (chat/invite system)
Core Logic
Matching algorithm based on:
Location similarity
Time overlap
Prioritizes:
Same or nearby routes
Minimal deviation
Sample Use Case

A student traveling from hostel to Katpadi at 5 PM creates a ride.
Other students with similar routes and timing can join, reducing cost and improving coordination.
